【游戏名称】
堆雪人 - H5纯净版

【游戏介绍】
这是一款轻松有趣的接物游戏。玩家控制一个可爱的雪人，在冬日雪景中接住从天而降的雪球、头部和礼物等物品来获得分数。游戏采用纯H5技术开发，无需安装，打开即玩。随着游戏进行，难度会逐渐增加，考验玩家的反应速度和判断能力。

【玩法说明】

障碍：
- 炸弹（黑色圆球带×标记）：接到会失去一颗生命值
- 游戏共有3颗生命值，失去所有生命值后游戏结束
- 随着游戏进行，物品掉落速度会越来越快

操作：
- 电脑端：移动鼠标左右控制雪人位置
- 移动端：手指在屏幕上左右滑动控制雪人移动
- 雪人会平滑跟随鼠标/手指位置移动

目标：
- 接住雪球（白色大球）：获得10分
- 接住头部（白色球带五官）：获得20分
- 接住礼物盒（金色礼物盒）：获得50分
- 避开炸弹，保护生命值
- 尽可能获得更高的分数

Tips：
- 优先接取礼物盒，分数最高
- 注意观察物品掉落轨迹，提前移动到位
- 炸弹出现频率较高，要时刻保持警惕
- 游戏难度会逐渐提升，保持专注
- 生命值显示在右上角，注意剩余数量

【技术介绍】
本游戏采用纯HTML5 Canvas技术开发，使用原生JavaScript实现游戏逻辑。游戏特点包括：
- 响应式设计，自适应不同屏幕尺寸
- 流畅的动画效果和平滑的移动控制
- 支持鼠标和触摸双重操作方式
- 动态难度调整系统，游戏体验更具挑战性
- 精美的渐变色彩和可爱的雪人造型
- 无需任何插件或框架，纯净H5实现
